## Technology Stack

- **Platform:** GitHub Pages
- **Generator:** Jekyll
- **Theme:** Custom styling based on Minima
- **Languages:** HTML, CSS, Markdown

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# View at http://localhost:4000
```
